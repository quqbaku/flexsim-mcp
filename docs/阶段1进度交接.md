## 3. 关键行为说明

### `evaluate()` 的语义
当前 FlexSimPy SDK 的 `Controller.evaluate()` 执行**节点路径求值**，不是任意 FlexScript 代码。

可用：
- `MAIN:/project/exec/time` — 获取仿真时间

当前 SDK 不接受：
- `return time();`
- `getmodeltime()`

**保护机制**：当传入脚本字符串且返回 `None` 时，会抛出明确错误。
